## Usage

## Development

```
git clone https://github.com/nielse63/node-scripts.git
cd node-scripts
nvm use
npm ci
```

After making desired changes and writing the required tests:

```
npm run lint
npm run build
npm test
```

The commit and push the changes to your branch and open a new pull request.
